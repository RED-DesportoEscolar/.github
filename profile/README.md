# RED Desporto Escolar — Recursos Educativos Digitais

Plataforma de recursos educativos digitais para o desenvolvimento da aptidão física e motora no Desporto Escolar, desenvolvida para a Direção-Geral da Educação (DGE).

---

## Módulos

| Módulo | Descrição |
|--------|-----------|
| **ACT.Ativate** | Atividades de aquecimento e ativação física |
| **ACT.Desafiate** | Jogo de desafios motores |
| **ACT.Diagnostico** | Avaliação diagnóstica das capacidades físicas |
| **ACT.Exercicio** | Exercícios com deteção de movimento em tempo real |
| **ACT.FactosCuriosos** | Conteúdo educativo sobre desporto e aptidão física |
| **ACT.TestaOteuConhecimento** | Quizzes de conhecimento desportivo |
| **ACT.VisitaVirtual** | Visita virtual 360º a espaços desportivos |
| **Plataforma** | Backend, gestão de utilizadores, sessões e badges |

---

## Lotes

A plataforma está organizada em 4 lotes de modalidades desportivas:

- **Lote 1** — Desportos Coletivos
- **Lote 2** — Desportos Individuais
- **Lote 3** — Desportos de Raquetes
- **Lote 4** — Desportos Gímnicos e Rítmico-Expressivos
